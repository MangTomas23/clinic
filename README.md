# Clinic

A simple desktop application made for small clinics.

## Features

* Ribbon style
* Patient's Record
* History record of patient
* Printing 
  * Patient's Data Sheet
  * Inventory
  * History record
  * Bills
  * Receipt
* Payment
* Simple Inventory
* Backup and restore database
